# init-vite

```bash
git clone git@github.com:nestaweb/init-vite.git ~/init-vite
```

To create a new project in your current folder run :

```bash
~/init-vite/init.sh <project-name>
```

After the first run you can do :

```bash
init-vite <project-name>
```

Update init-vite :

```bash
init-vite --update
```
